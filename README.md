# 🚨 Women Safety & Emergency Platform

A web-based safety application designed to provide **instant help during emergency situations**. This platform enables users to send SOS alerts, access emergency contacts, and simulate location-based assistance through an intuitive and responsive interface.

---

## 🚀 Live Demo

🔗 https://wseplatform.netlify.app/

---

## 📌 Features

- 🚨 **SOS Alert System** – Quickly trigger emergency alerts  
- 📍 **Location Sharing (Concept-based)** – Simulated real-time location tracking  
- 📞 **Emergency Contacts** – Quick access to important contacts  
- 🖥️ **Responsive Design** – Works seamlessly on mobile & desktop  
- 📊 **Dashboard Interface** – Structured layout for alerts and user interaction  
- ⚡ **Real-Time UI Updates** – Dynamic content updates using JavaScript  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3 (Flexbox & Grid)  
- JavaScript (DOM Manipulation)
